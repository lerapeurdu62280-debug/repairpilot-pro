## ❓ FAQ

**L'IA est-elle obligatoire ?**  
Non, le logiciel fonctionne parfaitement sans clé API. L'IA est un bonus optionnel.

**Mes données sont-elles en sécurité ?**  
Oui. Toutes vos données sont stockées localement sur votre machine. Rien n't envoyé sur internet sauf si vous utilisez l'IA.

**Puis-je utiliser le logiciel sur plusieurs PC ?**  
La licence est nominative (1 PC). Contactez-nous pour une licence multi-postes.

**Y a-t-il des mises à jour automatiques ?**  
Oui, le logiciel vérifie et installe les mises à jour automatiquement.

**Quelle est la politique de remboursement ?**  
Essai gratuit 30 jours disponible. Aucun remboursement après activation.
